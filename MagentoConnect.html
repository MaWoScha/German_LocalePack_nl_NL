Niederländisches Sprachpaket für Magento Commerce

(Magento Community Modules - Dutch (Netherlands) Language Pack)

(Paquete de idioma holandés para Magento Commerce)

Important note on Magento Version 1.9

Magento Inc. has pre-installed with the 1.9 private, semi-automatically translated, language pack. Who wants to use the LocalePack_nl_NL must first delete the app/locale/nl_NL folder so that this language pack can be installed.

Belangrijke opmerking over Magento Versie 1.9

Magento Inc. heeft vooraf geïnstalleerd met het 1.9 private, semi-automatisch vertaalde taalpakket. Wie LocalPack_nl_NL wil gebruiken, moet eerst de map app/locale/nl_NL verwijderen, zodat dit taalpakket kan worden geïnstalleerd.

Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. hat mit der 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_nl_NL benutzen möchte, muss vorher den Ordner app/locale/nl_NL löschen, damit dieses Sprachpaket installiert werden kann.

Nota importante acerca Magento Versión 1.9

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_de_DE, primero debe eliminar la carpeta app/locale/nl_NL para que el paquete de idioma se puede instalar.
